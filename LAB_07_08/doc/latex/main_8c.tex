\doxysubsection{Detailed Description}
Program that implements the State Machine of a vending machine. 

The program accepts a subset of coins and allows the user to browse through the available products, buy one product and return the credit. The inputs are push-\/buttons and the output is done via U\+A\+R\+T/\+Terminal. It was implemented using the board Nordic nrf52840-\/dk. The state machine is on the next Statechart.
